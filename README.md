# PetPaws Veterinary Hospital Management System

PetPaws is a web-based veterinary hospital management system designed to automate key clinic operations such as managing pets, appointments, and user roles.

## Tech Stack

### Frontend
-React.js

-Tailwind CSS


### Backend
-Node.js

-Express.js

-MySQL


## Features
-JWT based authentication

-Role-based access control

-Pet record management

-RESTful API backend

-Notification system

-Online appointment booking

-Medical history tracking
